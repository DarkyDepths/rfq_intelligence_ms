What this blueprint defines — the “how”
Every file to create or modify, every class and function responsibility, the exact data flow from package input to PackageParseEnvelope output, and the phased implementation sequence. All contracts, schemas, detection rules, and canonical section definitions are frozen in the Implementation Pack v1.0 — this blueprint does not redefine them. It maps how to build the code that produces them.

Convention note: This follows the established repo convention. Implementation Pack = the “what” (contracts, schemas, frozen shapes). Implementation Blueprint = the “how” (files to create, module structure, implementation sequence). Same convention as workbook_parser_implementation_pack_v*.html / workbook_parser_blueprint_v*.html.
Critical boundary — internal envelope vs persisted artifacts
PackageParseEnvelope is an internal parser output — never persisted as its own artifact. PackageInventory and SectionRegistry are intermediate processing structures inside the envelope. Persisted artifacts remain exactly two: rfq_intake_profile and intelligence_briefing. The intake_service consumes the envelope. The briefing_service consumes the enriched intake. No new artifact types are introduced.

03 Scanners — Stage A

3.1 normalizers.py — system file detection

Count semantics from Pack v1.0 §04: all count fields (total_files, FolderEntry.file_count, file_extension_counts) are business counts excluding system files. files[] list includes everything (raw truth). total_files_raw = len(files). system_file_count tallied separately. root_files excludes system files.

SYSTEM_FILE_PATTERNS = {".ds_store", "thumbs.db", ".gitkeep", "desktop.ini"}
SYSTEM_DIR_PATTERNS = {"__macosx"}

def is_system_file(filename: str) -> bool:
    return filename.lower() in SYSTEM_FILE_PATTERNS

def is_system_dir(dirname: str) -> bool:
    return dirname.lower() in SYSTEM_DIR_PATTERNS

3.2 tree_scanner.py

class TreeScanner:
    def scan(self, root_path: Path) -> PackageInventory:
        """Stage A: Walk filesystem → PackageInventory.

        Per file:
        - is_system_file via normalizers.is_system_file()
        - is_mr_index via MR Index regex (Pack §06.3)
        - mr_number_in_filename via MR-NNN regex (Pack §06.5)
        - section_prefix via NN_ prefix regex (Pack §06.4)
        - root_role for depth=0 non-system files (Pack §04.1):
            "mr_index_root" | "internal_review_file" | "unclassified_root_extra"

        Per folder:
        - number_prefix via NN- regex (Pack §06.2)
        - file_count = business count (excludes system files)

        PackageInventory:
        - total_files = business count
        - total_files_raw = all files including system
        - root_files = depth 0, excludes system files
        - file_extension_counts = business counts
        - system_file_count = count of system files
        """

05 Soft-Fail Contract — Three Distinct Outcomes

Same pattern as workbook parser _try_extract
None = section absent → extractor not called, envelope field is None.
Failed result = section present but extractor crashed → error issue captured, envelope field is None, error in parser_report.errors.
Success result = normal extraction → data assembled into envelope.
def _try_extract(self, canonical_key, extractor, inventory, registry):
    section = self._find_section(registry, canonical_key)
    if section is None:
        return None
    try:
        return extractor.extract(inventory, registry)
    except Exception as exc:
        logger.error("Extraction failed for '%s': %s", canonical_key, exc)
        return _make_failed_issue(canonical_key, exc)

06 Orchestrator — parser_orchestrator.py

PARSER_VERSION = "package-parser-v1.0"

class PackageParserOrchestrator:
    def parse(self, package_path: str, rfq_id: str) -> dict:
        path = Path(package_path)

        # Stage A: Package Scan
        if path.suffix.lower() == ".zip":
            inventory = ZipScanner().scan(path)
        elif path.is_dir():
            inventory = TreeScanner().scan(path)
        else:
            raise ValueError(f"Unsupported input: {package_path}")

        # Stage B: Structure Recognition
        identity = IdentityExtractor().extract(inventory)
        registry = SectionClassifier().classify(inventory)

        # Stage C: Targeted Extraction
        standards   = _try_extract("applicable_standards", StandardsExtractor(), ...)
        bom         = _try_extract("description_bom", BomExtractor(), ...)
        rvl         = _try_extract("rvl", RvlExtractor(), ...)
        sa175       = _try_extract("sa175_forms", Sa175Extractor(), ...)
        compliance, deviation = _try_extract_compliance(...)

        # Cross-checks (Pack §08)
        cross_checks = run_cross_checks(identity, bom, rvl, compliance, deviation, registry, inventory)

        # Stage D: Assembly
        return build_envelope(rfq_id, inventory, identity, registry,
                              standards, bom, rvl, sa175, compliance, deviation, cross_checks)

07 Assembler — build_envelope()

def build_envelope(...) -> dict:
    """Same pattern as workbook assembler:
    1. Collect issues from all extractors
    2. Compute overall status (parsed_ok / parsed_with_warnings / failed)
    3. Build StageReport per stage
    4. Assemble PackageParseEnvelope (Pack §04.11)
    5. Return dataclasses.asdict(envelope)

    input_path is set to None (do not persist local paths)."""
